Cat Drop

Puzzles
Help the little fallen Kitty Cat to land in the right platform! How to play: Click on the boxes, blocks etc... to remove them from the screen. All boxes except the grass must all be removed for you to reach the next level. Use the set blocks and props however you can to get the Little Cat land safe to the grass aera. You can reset each level if you make a wrong move. Try to complete all 28 different levels to resolve in this fun physics puzzle game!
